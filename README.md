# SarvAI (सर्व.AI)

## Overview
ServAI is a Full stack PERN-based web application to meet all your AI realted need onto 1 place, This website offers the opportunity to create articles, generate blog-titles, generate images, remove backgrounds/objects and engage in the exciting new community and like all the posts you find intresting.
<br/>
So Go ahead and try ServAI to meet all you expectations in just one place.

### Features:




-**All-in-One AI Workspace**: Multiple AI services (image, text, PDF, resume, etc.) in one place

-**Modern Full-Stack Web App**: Built with React, Express, PostgreSQL, and Node.js (PERN stack)

-**User Authentication**: Secure sign up, login, and profile management using Clerk

-**Plan-based Access**: Premium and standard feature tiers

-**Image AI Tools**: Background removal, object removal, and more via Cloudinary

## Technologies Used:
- **Frontend**: React, JavaScript, React-Router, TailwindCSS, ContextAPI
- **Backend**: Node.js, Express, Axios, Cors
- **Database**: PostgreSQL, Neon
- **API**: Gemini, ClipDrop
- **Authentication**: Clerk's JWT Token-based Authentication

## Installation & Setup

### Prerequisites:
- Node.js
- PostgreSQL (or Neon)
- npm

### 1. Clone the repository:
```bash
git clone https://github.com/Omsh24/ServAI.git

```
### 2. Install dependencies
1. Server
   ```bash
   cd server
   npm install
   ```
2. Frontend
   ```bash
   cd frontend
   npm install
   ```
   
### 3. Configuration
backend: Create a .env file in the backend directory and add your MongoDB URI and any necessary environment variables.
**Sample env file:**
-DATABASE_URL = <br/>
-CLERK_PUBLISHABLE_KEY = <br/>
-CLERK_SECRET_KEY = <br/>
-GEMINI_API_KEY = <br/>
-CLIPDROP_API_KEY = <br/>
-CLOUDINARY_CLOUD_NAME = <br/>
-CLOUDINARY_API_KEY = <br/>
-CLOUDINARY_API_SECRET = <br/>

### 4. Run the project
1. Server
   ```bash
   cd server
   npm run server
   ```
2. Frontend
   ```bash
   cd frontend
   npm run dev
